# Task 1: Hello World Jenkins Pipeline

## Objective
Create a basic Jenkins pipeline that demonstrates:
- Pipeline structure
- Basic stages
- Environment variables
- Workspace operations

## Pipeline Stages
1. **Hello World** - Print greeting message
2. **Show Time** - Display current date and time
3. **Build Info** - Show build details
4. **Workspace** - List workspace contents

## How to Use
1. Create a new Jenkins pipeline job
2. Point it to this repository
3. Run the pipeline
4. Check console output

## Expected Output
Hello from Jenkins Pipeline!
Current Date: 2024-01-01 12:00:00
Build Number: 1
Build URL: http://jenkins:8080/job/task1/1/
Workspace Contents: Jenkinsfile README.md
